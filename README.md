# Kanban Board API

## Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- npm

## Installation

1. Clone the repository
```bash
   git clone <https://github.com/tarannumtasfia/kanban_board_api.git>
   cd kanban_board_api
```

2. Install dependencies
```bash
   npm install
```

## Running the Project

```bash
npm start
```

Server will run at `http://localhost:3000`

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Get all tasks |
